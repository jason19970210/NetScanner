# NetScanner
Automatic scan the network hosts and get operation system, MAC address ... etc.




### System Requirements
- Only for MacOS & Linux due to the limitation of packages

### Requirements
- Python3
- Pip Packages
    - nmap
    - nmap3
    - netifaces
    - prettytable
    - inquirer

### Usage
**Just run the network_scanner.py** then following the steps ~~
