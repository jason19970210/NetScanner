### System Requirements
- Only for MacOS & Linux due to the limitation of packages

### Requirements Install
```bash
$ pip3 install -r requirements.txt
```

### Usage
Clone the repo and run the net_scanner.py as root
```bash
$ sudo python3 net_scanner
```
- No arguments needed
